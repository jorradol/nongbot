# น้องบอท (NongBot) 🍊 — AI Sales Content Assistant

**NongBot (น้องบอท)** เป็นแพลตฟอร์มผู้ช่วยปั้นโพสต์โฆษณาขายของออนไลน์ด้วยปัญญาประดิษฐ์ (AI-powered Ads copywriting platform) ที่ถูกพัฒนาขึ้นมาเพื่อแก้ปัญหากระดูกชิ้นโตของบรรดาพ่อค้าแม่ค้ายุคดิจิทัล เช่น การคิดแคปชันไม่ออก, เขียนข้อความขายไม่เก่ง, ไม่รู้พิกัดแฮชแท็กติดเทรนด์, หรือต้องการความรวดเร็วในการเปิดตัวลงโพสต์สินค้า

ด้วยหลักการสแกนด้วยรูปภาพเดียวบวกการคุยโต้ตอบเป็นขั้นตอนอย่างอบอุ่น (Step-by-step Conversation) จากน้องบอท ระบบจะคลอดพาดหัว Hook, คำอธิบายประเด็นเด่น, คีย์เวิร์ดกระตุ้นความรู้สึก และคอลเลกชันแฮชแท็กสำเร็จรูปเหมาะสำหรับการนำไป Copy -> Paste ลงช่องทางจำหน่ายหลัก (Facebook, TikTok, Instagram, Shopee, Lazada) ได้ทันที

---

## 🚀 คุณสมบัติเด่น (Core Features)

1. **โหมดล็อกอินที่ยืดหยุ่น (Authentication)**
   * ขับเคลื่อนผ่าน **Firebase Authentication** ได้แก่ การล็อกอินด้วย Google Account และ **โหมดทดลองใช้ทันที (Demo Mode)** สำหรับกรณีต้องการเริ่มทดสอบประเมินระบบแบบปราศจากขั้นตอนสมัคร
   * มีความปลอดภัยสูงด้วยระบบสิทธิ์ป้องกัน Firestore Security Rules

2. **ระบบสแกนวิเคราะห์ภาพด้วย AI (AI Visual Analysis)**
   * สกัดคีย์สไตล์, หมวดผลิตภัณฑ์, จุดเด่นของภาพ, และกลุ่มหัวหอกลูกค้าเป้าหมายทันทีด้วยคลังอัจฉริยะ **Gemini 3.5-flash** ตรวจจับภาพบวกบรีฟข้อมูล

3. **แชทพรีเซนต์คำตอบด่วนโต้ตอบด่วน (Step-by-step Chat Assistant)**
   * ปฏิวัติฟอร์มพิมพ์ข้อมูลแบบเก่าที่ยาวจนหน้าเบื่อ เป็นการตอบสารคดีทักทายกับน้องบอทอย่างเป็นระบบกิริยา
   * มีคีย์บอร์ดตอบด่วน (Quick Preset Replies) ที่แค่เคาะก็กรอกสเกลราคาและโปรโมชั่นเด็ดพ่วงส่งข้อมูลทันทีสไตล์ One-Click

4. **การรังสรรค์ก๊อปปี้โฆษณาระดับเซียน (Content Generation)**
   * พุ่งกลุ่มเป้าหมายสไตล์อิสระ 5 โทนเสียง ได้แก่: **มืออาชีพ**, **กันเอง**, **ปิดการขายแรงแบบเร่งด่วน**, **สายมินิมอล**, และ **TikTok Viral**
   * จัดโครงสร้างคำโฆษณาเป็นหมวดหมู่ (พาดหัวหลัก, แนะนำฟีเจอร์พรีเมียม, ประตุ้นอารมณ์คลั่งรัก, เรียกร้องสั่งซื้อ CTA, และแฮชแท็ก)
   * 1-Click Copy Clipboard นำข้อความก๊อปปี้ทั้งหมดไปแปะในหน้าขายของ

5. **แดชบอร์ดสรุปประสิทธิภาพส่วนหลังบ้าน (Admin Dashboard)**
   * รวบรวมแง่มุมสถิติระบบทั้งหมด: ยอดรวมจำนวนผู้ใช้งาน, ชิ้นงานโพสต์ทั้งหมด, ปริมาณผู้ใช้งานแอ็กทีฟ, อัตราเฉลี่ยเหมาจ่ายเครดิต
   * แผงควบคุมเติมเหรียญ (Credits Incremetor) สำหรับบริหารเติมเครดิตผู้ค้าขายรายหัวได้อย่างปลอดภัย

## 🛠️ โครงสร้างเทคโนโลยี (Tech Stack)

* **Frontend:** React 19, TypeScript, Tailwind CSS v4, Motion Transitions
* **Backend:** Node.js Express, `@google/genai` (Gemini-3.5-flash)
* **Database & Auth:** Firebase Web Client, Firestore Database Cloud

## 📝 ตัวแปรสภาพแวดล้อม (.env.example)

แอปพลิเคชัน NongBot ถูกตั้งค่าให้เป็นระบบ **Zero-Configuration** สำหรับ Firebase เนื่องจากไฟล์ตั้งค่า credentials ทั้งหมดถูกดึงแบบไดนามิกจากระบบ `firebase-applet-config.json` ที่ระบบจัดเตรียมไว้อย่างปลอดภัย ส่วน Gemini API Key จะถูกป้อนฝั่งหลังบ้านอย่างสมบูรณ์ไร้รอยรั่วไปยังฝั่งบราวเซอร์:

```env
# Gemini API Key ป้อนเข้ารหัสฝั่งเซิร์ฟเวอร์เท่านั้น
GEMINI_API_KEY="AIzaSy..."

# ลิงก์ URL เปิดพอร์ตให้บริการ
APP_URL="http://localhost:3000"
```
